# Ghost Diving Spain - Reporte de Redes de Pesca

Página web para reportar redes de pesca fantasma encontradas durante actividades subacuáticas, inspirada en Ghost Diving Spain.

## Características

- Formulario completo para reportar redes de pesca
- Almacenamiento de datos en localStorage y CSV descargable
- Interfaz responsive y moderna
- Estadísticas en tiempo real
- Modal para visualizar reportes recibidos
- Funcionalidad de búsqueda y filtrado
